[,ruby]
----
puts 'Hello, World!'
def greet(name)
  puts "Hello, #{name}!"
end
----

This should parse as JavaScript:

[,javascript]
----
console.log('Hello, World!');
function greet(name) {
  console.log(`Hello, ${name}!`);
}
----

Regular code block without language:

----
echo "Hello, World!"
----
